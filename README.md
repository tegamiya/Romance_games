# Romance_games

## 概要
C言語+DXライブラリで作成した恋愛ノベルゲーム  
ストーリーはネットに落ちてたものを改変した  
ストーリー的には未完成（同じ作業の繰り返しで終わるため、技術力向上の意味が無いとみなし作成を中止した）

## 参考にしたもの
http://www.usamimi.info/~guiprog/game_programming/gameProgramming.pdf
